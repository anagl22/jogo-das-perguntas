Como jogar

Escolha número de perguntas e temporizador. O tempo máximo por pergunta é 60s e o mínimo é 0s. Quanto menos tempo levar, mais pontos ganha.

  • Use 1–4 para responder, Enter para próxima.
  • Lifeline 50/50 disponível (remove 2 alternativas).
  • Ranking por número de perguntas (cada N tem seu ranking separado).
Fórmula de pontos (por pergunta correta):
points = round( base × (1 + (MAX - t)/MAX) × (1 + N/20) )
Onde: base = 100, MAX = 60s, t = tempo que você levou (s), N = número de perguntas.
